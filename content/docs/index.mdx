## Introduction

ink-ui is a terminal UI component library built with [Ink](https://github.com/vadimdemedes/ink) for the web using ink-web and xterm.

## Installation

Install ink-ui and its peer dependencies:

```bash
bun add ink-web xterm
```

## Quick Start

Create a terminal component:

```tsx
"use client";

import { InkXterm, Box, Text } from "ink-web";
import "ink-web/css";
import "xterm/css/xterm.css";

export const Terminal = () => (
  <InkXterm focus>
    <Box flexDirection="column">
      <Text color="green">Hello from Ink!</Text>
    </Box>
  </InkXterm>
);
```

Then dynamically import it in your page (SSR must be disabled for terminal components):

```tsx
"use client";

import dynamic from "next/dynamic";

const Terminal = dynamic(() => import("./terminal"), {
  ssr: false,
});

export default function Home() {
  return <Terminal />;
}
```
