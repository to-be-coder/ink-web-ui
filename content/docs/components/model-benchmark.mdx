## Features
- 6 AI models racing through a benchmark with animated progress bars
- Real-time tokens/sec sparkline visualization per model
- Staggered start with warm-up latency simulation
- Auto-switch to ranked leaderboard on completion
- Medal indicators for top 3 finishers
- Accuracy scores, TTFT, and cost comparison

## Keybindings
- `space` — Pause/resume
- `v` — Toggle between race view and leaderboard
- `r` — Restart benchmark
