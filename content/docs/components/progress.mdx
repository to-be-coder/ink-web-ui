## Features

- **Spring-based animation** — smooth transitions using spring physics (stiffness + damping)
- **Half-block precision** — uses `▌` character for sub-character resolution
- **Multiple bar styles** — solid fill, gradient blend, and custom characters (`▰▱`)
- **Color gradients** — smooth color interpolation between start and end colors
- **Percentage display** — current progress shown alongside each bar
- **Completion indicator** — checkmark appears when a task finishes

## Keybindings

- `space` pause/resume progress
- `r` reset all bars to zero
