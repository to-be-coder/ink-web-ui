## Features
- Side-by-side split layout with original (left) and modified (right) columns
- Red removals on the left, green additions on the right
- Dual line number gutters per side
- Empty slots for added/removed lines that don't have a counterpart
- File tabs with addition/deletion counts
- Scrollable with truncation for long lines

## Keybindings
- `f` / `tab` — Switch files
- `j` / `k` — Scroll
