## Message Parts
- **Text** — AI reply, always visible
- **Reasoning** — Thinking trace, collapsed by default, shows "Thought for Xs"
- **Tool Call** — File reads, searches, edits. Collapsed by default with status icon
- **Code** — Syntax-highlighted code block with line numbers and file path
- **Source** — Citation references, inline compact

## Keybindings
- `j` / `k` — Navigate between message parts
- `enter` / `space` — Toggle collapse on reasoning and tool calls
