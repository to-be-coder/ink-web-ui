## Features
- Phase-based layout (Plan → Research → Implement → Validate → Finalize)
- Animated playthrough with circle-fill loading indicator
- Parallel agents with "concurrent" label
- Continuous vertical connector line colored by status
- Agent titles: gray (pending) → blue (running) → green (done)
- Per-step duration display
- Toggle expand/collapse all phases

## Keybindings
- `enter` / `space` — Toggle expand/collapse
- `r` — Restart animation
