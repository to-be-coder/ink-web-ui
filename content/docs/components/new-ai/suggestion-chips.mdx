## Features
- Categorized suggestions: follow-up, explore, action, refine
- Category legend with color coding
- Staggered fade-in animation for progressive reveal
- Bordered highlight on cursor focus
- Selection confirmation with visual feedback
- Multiple suggestion sets with cycling

## Keybindings
- `j` / `k` — Navigate suggestions
- `⏎` — Select suggestion
- `n` — Next suggestion set

## Research Context
Inspired by **LlamaIndex Chat UI** (`SuggestedQuestions`, `StarterQuestions`) and **Vercel AI Elements** (`Suggestion`). Categorized suggestions with visual differentiation goes beyond simple prompt chips that most libraries offer.
