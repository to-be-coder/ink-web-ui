## Features
- Circle-fill loading animation (○ → ◔ → ◑ → ◕ → ●)
- Continuous colored connector lines (green done, red error, primary running)
- Expandable log details per step
- Auto-animated progression with elapsed timer
- Error handling with test failure + fix flow

## Keybindings
- `d` — Toggle log details
- `space` — Pause/resume
- `r` — Restart
