## Features
- Tab toggle between Response and Sources views
- Inline numbered citation references `[1,2]` within response text
- Collapsible source details (URL, snippet)
- Navigate and expand individual sources

## Keybindings
- `v` — Toggle between Response and Sources views
- `j` / `k` — Navigate sources (in Sources view)
- `enter` / `space` — Expand/collapse source details
