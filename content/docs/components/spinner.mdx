## Features

- **11 spinner styles** — Line, Dot, MiniDot, Jump, Pulse, Points, Globe, Moon, Meter, Hamburger, Ellipsis
- **Independent frame timing** — each spinner type runs at its own FPS for authentic animation
- **Contextual display** — spinners shown alongside task labels for real-world usage
- **Pause/resume** — space key freezes all animations

## Keybindings

- `↑/↓` navigate spinner list
- `space` pause/resume animation
