## Features

- **Rotating gradient** — time-based 2D rotation transform creates sweeping color bands
- **12-color rainbow** — smooth interpolation across purple, red, orange, yellow, green, cyan, blue
- **Half-block rendering** — uses `▀` characters for double vertical color resolution
- **Variable speed** — three speed settings for the rotation animation
- **60 FPS** — smooth animation targeting 60 frames per second

## Keybindings

- `space` pause/resume animation
- `1` slow speed
- `2` normal speed
- `3` fast speed
