## Features

- **Multiple timers** — manage several independent countdown timers (Pomodoro, Break, Meeting, Quick)
- **Start/stop/toggle** — space key controls the selected timer
- **Visual progress bar** — color changes from blue to yellow to red as time runs out
- **Duration adjustment** — increase or decrease timeout by 30-second increments
- **Timeout detection** — timers auto-stop and show "done" when they reach zero
- **Batch controls** — start all or stop all timers at once

## Keybindings

- `↑↓` / `j/k` select timer
- `space` start/stop selected timer
- `r` reset selected timer
- `←→` / `+/-` adjust duration by 30s
- `s` start all timers
- `x` stop all timers
