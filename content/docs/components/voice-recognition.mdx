## Features

- **Animated waveform** with 16-bar frequency visualization and amplitude-reactive coloring
- **Real-time transcription** using the browser's Web Speech API
- **Interim results** shown as you speak, finalized when a phrase completes
- **Demo mode** simulates the full experience without microphone access
- **Duration timer** tracks recording time
- **Transcript history** with the 5 most recent entries visible

## Keybindings

- `space` start/stop listening
- `d` start/stop demo simulation
- `c` clear transcript history
