## Features

- **Sequential installation** — packages install one at a time with random 100-500ms delays
- **Animated spinner** — braille dot spinner while each package installs
- **Progress bar** — visual progress indicator with fill/empty blocks
- **Status tracking** — checkmarks for installed packages, spinner for current, counter for progress

## Keybindings

- `space` start installation
- `r` reset
