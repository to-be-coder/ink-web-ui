## Features

- **Procedural starfield** — depth-based grayscale gradient with random noise for star-like appearance
- **Horizontal scrolling** — continuous parallax scrolling effect via frame counter offset
- **Half-block rendering** — uses `▀` characters for double vertical pixel density
- **60 FPS** — smooth animation targeting 60 frames per second

## Keybindings

- `space` pause/resume animation
