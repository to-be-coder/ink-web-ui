## Features

- **Thinking steps** showing the agent's reasoning before acting
- **Tool calls** with input arguments and returned results
- **Streaming response** rendered character by character
- **Animated spinners** on active steps with checkmarks on completion
- **Multiple demo queries** to cycle through with `n`

## Keybindings

- `space` run the agent
- `n` next query
- `r` replay current query
