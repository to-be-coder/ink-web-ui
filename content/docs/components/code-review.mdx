## Features
- Animated scanning → reviewing → complete phases with spinner
- Inline annotations appear progressively on diff lines
- Severity levels: critical, warning, suggestion, praise
- Suggested code fixes shown inline with green `+` prefix
- Accept or dismiss individual annotations
- File tabs with addition/deletion counts
- Scrollable diff with line numbers and +/- prefixes

## Keybindings
- `f` / `tab` — Switch files
- `j` / `k` — Scroll diff and navigate annotations
- `a` — Accept focused annotation
- `x` — Dismiss focused annotation
