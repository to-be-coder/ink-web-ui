## Features
- Reply-to threading with quoted message preview
- Message reactions with counts
- Read receipt indicators (sent, delivered, read)
- Navigable message list with cursor
- Streaming text with spinner
- Inline code and code block formatting
- Token counts and model attribution

## Keybindings
- `j` / `k` — Navigate messages
- `r` — Replay animation
