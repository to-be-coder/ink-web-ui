## Features
- Minimal prompt-style user messages with `>` prefix
- Inline thinking/reasoning shown in italic dim text
- Toggle thinking visibility with `t` key
- Streaming text animation
- Indented code-like lines highlighted
- Footer metadata (model, tokens, timestamp)

## Keybindings
- `r` — Replay animation
- `t` — Toggle thinking visibility
