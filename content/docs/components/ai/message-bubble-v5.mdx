## Features
- Background-colored chat bubbles — dark navy for user, dark blue for assistant
- Full-width padded content blocks with consistent bubble width
- Inline code highlighted with contrasting background
- Indented code lines in secondary color
- Streaming text animation within the bubble
- Token counts, timestamps, and model attribution in header
- Role-colored accent on header text

## Keybindings
- `r` — Replay animation
