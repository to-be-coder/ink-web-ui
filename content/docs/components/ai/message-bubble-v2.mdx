## Features
- Vertical accent bar per message colored by role
- Streaming text animation with cursor
- Inline code highlighting with backticks
- Numbered list formatting
- Token counts, timestamps, and model attribution
- Auto-replay animation

## Keybindings
- `r` — Replay animation
