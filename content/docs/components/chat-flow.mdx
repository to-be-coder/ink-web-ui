## Features
- Full animated conversation replay showing the AI lifecycle
- **Thinking** — progressive line reveal with braille spinner, auto-collapses when done
- **Tool calls** — Read, Search, Edit, Write with color-coded tool names and results
- **Streaming text** — character-by-character typing with cursor
- **Code blocks** — line-by-line reveal with syntax coloring and line numbers
- Token count and cost tracking per message
- Auto-collapse thinking and tool calls after completion
- Multi-turn conversation with 2 user messages and 2 assistant responses

## Keybindings
- `j` / `k` — Scroll
- `e` — Expand/collapse all thinking and tool calls
- `r` — Replay animation
