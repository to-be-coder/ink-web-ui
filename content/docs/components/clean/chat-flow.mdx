## Features
- Animated conversation replay showing the AI lifecycle
- **Thinking** — progressive reveal with minimal spinner, auto-collapses when done
- **Tool calls** — Read, Edit, Write, Bash with color-coded names and results
- **Streaming text** — character-by-character typing with cursor
- Token count per message
- Auto-collapse thinking and tool calls after completion
- Multi-turn conversation with replay

## Keybindings
- `e` — Expand/collapse all thinking and tool calls
- `r` — Replay animation
