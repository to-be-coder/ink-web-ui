## Features

- **Two display modes** — Arabic numerals (`1/4`) and dots (`●∙∙∙`)
- **Keyboard navigation** — arrow keys, vim keys, and first/last shortcuts
- **Item list** — paginated content with numbered items
- **Dynamic padding** — empty rows preserved when last page is short

## Keybindings

- `←→` / `h/l` / `n/p` previous/next page
- `g` first page
- `G` last page
- `t` toggle between Arabic and dot display
