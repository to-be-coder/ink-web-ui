## Features

- **Live updates** every second with simulated CPU, memory, disk, and network stats
- **Progress bars** that change color based on usage (green, yellow, red)
- **Sortable process list** by CPU, memory, or name
- **Pause/resume** updates with `p`

## Keybindings

- `p` pause/resume
- `c` sort by CPU
- `m` sort by memory
- `n` sort by name
