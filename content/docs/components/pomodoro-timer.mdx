## Features

- **25/5 work/break cycle** with automatic phase switching
- **Visual progress bar** showing time remaining
- **Session counter** tracking completed work sessions and total focused minutes
- **Adjustable durations** for both work and break periods

## Keybindings

- `space` start/pause
- `r` reset timer
- `s` skip to next phase
- `↑` `↓` adjust work duration
- `+` `-` adjust break duration
