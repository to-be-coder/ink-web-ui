## Features

- **High-precision timing** — uses requestAnimationFrame for smooth centisecond display
- **Large digit display** — block characters render time in a prominent 3-row format
- **Lap tracking** — record split times with best/worst lap highlighting
- **Color coding** — green for best lap split, red for worst lap split
- **Lap viewport** — most recent 6 laps shown with overflow indicator

## Keybindings

- `space` start/stop
- `l` record lap
- `r` reset
