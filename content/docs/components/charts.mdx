## Features

- **Braille line chart** using Unicode Braille patterns (U+2800-U+28FF) for high-resolution 2D plotting with multiple series
- **Bar chart** using block characters for vertical bars with color coding by value
- **Sparkline** inline trend charts using `▁▂▃▄▅▆▇█` block elements with multiple series
- **Gauge** horizontal progress bars with color thresholds (green/yellow/red)
- **Auto-updating** data with pause/resume and reset controls
- **Tabbed view** to switch between all four chart types

## Keybindings

- `1` Line chart
- `2` Bar chart
- `3` Sparkline
- `4` Gauge
- `space` pause/resume auto-update
- `r` reset data
