## Features

- Braille-pattern spinner animation on active tasks
- Pipeline visualization with vertical flow connectors
- Auto-progressing task pipeline with cascading status
- Color-cycling animation on active task (primary/secondary blend)
- Status icons: checkmark (done), cross (error), circle (pending)
- Task detail annotations with context
- Completion summary with error count
- Pause/resume and restart controls

## Keybindings

- `space` - Pause/resume pipeline
- `r` - Restart all tasks
