## Features

- 2D cursor navigation across rows and columns
- Column sorting with direction indicators
- Multi-row selection with select all/clear
- Inline text filtering across all columns
- Status pills with semantic colors (running/stopped/pending)
- Zebra-striped rows for readability
- Viewport scrolling with position indicator
- Right-aligned numeric columns
- Badge showing selected count and active filter

## Keybindings

- `hjkl` or arrows - Navigate cells
- `s` - Sort by current column
- `space` - Toggle row selection
- `a` - Select all rows
- `x` - Clear selection
- `/` - Enter filter mode
