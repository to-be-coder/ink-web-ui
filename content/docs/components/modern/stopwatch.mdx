## Features

- Large block-character time display (MM:SS.CC)
- High-frequency animation using requestAnimationFrame
- Lap recording with split and total times
- Best lap highlighted in green, worst in red
- Scrollable lap history (6 most recent visible)
- Status indicator (Running/Paused/Ready)
- Color-coded display matching state
- Clean separator between time and laps

## Keybindings

- `space` - Start/stop
- `l` - Record lap (while running)
- `r` - Reset (while stopped)
