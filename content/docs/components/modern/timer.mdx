## Features

- 4 preset timers (Pomodoro, Break, Meeting, Quick)
- Color-shifting progress bars (blue to green, then warning/error)
- Real-time countdown with formatted display
- Time adjustment in 30-second increments
- Individual toggle and batch start/stop all
- Status indicators (running/paused/done)
- Running count display
- Reset individual timers

## Keybindings

- `j/k` or `up/down` - Select timer
- `space` - Toggle start/pause
- `r` - Reset current timer
- `left/right` or `+/-` - Adjust duration by 30s
- `s` - Start all timers
- `x` - Stop all timers
