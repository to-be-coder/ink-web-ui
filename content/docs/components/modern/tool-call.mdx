## Features

- Feed-style layout — each call is an independent block, not a pipeline
- Color-coded inverse tool badges (Read, Search, Edit, Write, Bash)
- Braille spinner on the active call
- Results shown inline when complete — no expand needed
- Calls appear one at a time, previous auto-completes
- Error state for failed calls with highlighted output

## Keybindings

- `r` - Restart demo
