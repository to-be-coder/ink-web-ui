## Features

- Unified diff format with +/- line markers
- Dual line number gutter (old and new)
- Color-coded additions (green) and deletions (red)
- Multiple file support with tab navigation
- File-level addition/deletion stats
- Collapsible unchanged regions
- Compact mode showing only changes
- Total change summary

## Keybindings

- `f` - Next file
- `↑/↓` - Scroll
- `d` - Toggle compact/full view
