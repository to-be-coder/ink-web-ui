## Features

- Clack-style guide bar with vertical flow
- Grouped options (Frontend, Backend, Database)
- Radio-style indicators with active highlight
- Inline filter with match count
- Description shown only for focused option
- Confirmation card with selected value
- Reset to start over

## Keybindings

- `j/k` or `up/down` - Navigate options
- `enter` - Confirm selection
- `/` - Enter filter mode
- `r` - Reset (after confirmation)
