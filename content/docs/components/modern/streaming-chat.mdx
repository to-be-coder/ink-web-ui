## Features

- Message bubbles with role labels (You / Assistant)
- Streaming text effect with progressive token display
- Braille spinner typing indicator while generating
- Token count per message and running total
- Markdown-lite rendering (bold, code, lists)
- Auto-playing conversation demo
- Scroll navigation

## Keybindings

- `↑/↓` - Scroll messages
- `r` - Restart conversation
