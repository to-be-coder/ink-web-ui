## Features

- Clack-style guide bar flow with diamond/circle symbols
- Multi-step groups with progress dots
- 5 field types: select, input, multiselect, confirm, note
- Inline validation with error states
- Summary review before submission
- Tab/Shift+Tab for field navigation
- Card container with rounded borders

## Keybindings

- `tab` / `shift+tab` - Navigate between fields
- `enter` - Confirm and advance
- `j/k` or `up/down` - Select options
- `space` - Toggle multiselect
- `y/n` - Quick confirm
- `left/right` - Toggle confirm value
