## Features

- Moon phase spinner animation on running tasks
- Emoji status icons for each state (sleeping, exploding, sparkles)
- Animated clock emoji in the header matching elapsed time
- Witty quips shown under each running task
- Color-transitioning progress bar with rocket indicator
- Expandable detail logs per task
- Playful completion messages with total run time
- Same controls as Task Pipeline v1

## Keybindings

- `d` - Toggle detail logs
- `space` - Pause/resume pipeline
- `r` - Restart all tasks
