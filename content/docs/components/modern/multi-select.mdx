## Features

- Clack-style guide bar with vertical flow
- Grouped options (Languages, Frameworks, Infrastructure, Tools)
- Checkbox-style toggle indicators
- Select all / clear all batch operations
- Inline filter with match count across label, group, and description
- Scrolling viewport with position indicator
- Selected count badge
- Confirmation view grouped by category

## Keybindings

- `j/k` or `up/down` - Navigate options
- `space` - Toggle selection
- `a` - Select all visible
- `x` - Clear all selections
- `enter` - Confirm
- `/` - Enter filter mode
- `r` - Reset (after confirmation)
