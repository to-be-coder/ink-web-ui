## Features

- Pill-style active tab with inverse highlight
- Dot indicators showing total tab count and position
- Scroll indicators for overflowing tab bar
- Tab reordering with Shift+arrow keys
- Create and close tabs dynamically
- Number key quick-jump (1-9)
- Pinned tabs (non-closable)
- Code preview with line numbers and separator

## Keybindings

- `h/l` or `left/right` - Switch tabs
- `1-9` - Jump to tab by number
- `n` - Create new tab
- `w` - Close current tab
- `H/L` or `Shift+arrows` - Reorder tabs
