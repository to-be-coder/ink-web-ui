## Features

- Segmented bar showing system prompt, conversation, and tool result usage
- Color transitions as usage grows (green → yellow → red)
- Warning threshold indicator at 80%
- Breakdown legend with per-section token counts
- Sparkline showing usage trend over time
- Two view modes: overview bar and detailed breakdown
- Simulated context growth animation

## Keybindings

- `v` - Toggle between overview and breakdown view
- `r` - Restart simulation
