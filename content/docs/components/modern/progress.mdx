## Features

- Spring physics animation for smooth progress transitions
- Sub-character precision with 1/8th block characters
- Gradient color interpolation (per-task custom colors)
- Inline sparkline charts showing progress velocity
- Individual task tracking with completion checkmarks
- Aggregate total progress bar with separator
- Elapsed time counter
- Pause/resume and reset controls

## Keybindings

- `space` - Pause/resume all tasks
- `r` - Reset progress to zero
