## Features

- Severity-coded permission cards (info, warning, danger)
- Color-coded borders and icons by risk level
- Allow, deny, and allow-for-session controls
- File path details for file operations
- Decision history log with status badges
- Auto-queued permission requests
- Compact resolved state

## Keybindings

- `y` - Allow action
- `n` - Deny action
- `s` - Allow for session
- `r` - Restart demo
