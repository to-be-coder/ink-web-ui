## Features

- 3 pagination display modes: dots, numbers, progress bar
- Toggle between styles with `t` key
- 25 infrastructure services across 5 pages
- Status badges with semantic colors (active/idle/error)
- Number key quick-jump to specific pages
- First/last page navigation
- Clean separator between content and pagination
- Service count display

## Keybindings

- `h/l` or `left/right` - Previous/next page
- `g/G` - Jump to first/last page
- `1-5` - Jump to specific page
- `t` - Toggle pagination style
