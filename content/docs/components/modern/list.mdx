## Features

- Fuzzy matching filter with instant results
- Multi-select with checkbox indicators
- Category badges with semantic colors (core, overlay, data, nav, feedback, display)
- Viewport scrolling with overflow indicators
- Select all / clear selection
- Status messages with auto-clear
- Vim-style navigation (j/k, g/G)
- Item count showing filtered vs total

## Keybindings

- `j/k` or `up/down` - Navigate items
- `/` - Enter filter mode
- `space` - Toggle selection
- `enter` - Open/activate item
- `a` - Select all
- `x` - Clear selection
- `g/G` - Jump to first/last
