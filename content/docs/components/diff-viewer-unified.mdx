## Features
- Unified single-column diff with `+/-` prefixes
- Word-level change highlighting (bold + underline on changed segments)
- Collapsible hunk sections with `▸/▾` indicators
- Hunk headers showing line range info
- File tabs with per-file stats
- Clean, minimal layout

## Keybindings
- `f` / `tab` — Switch files
- `j` / `k` — Scroll
- `c` — Collapse/expand all hunks
