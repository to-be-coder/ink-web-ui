# ink-web-ui

A component library built on top of [ink-web](https://github.com/cjroth/ink-web) by [Chris Roth](https://github.com/cjroth). Renders terminal UI components in the browser using [Ink](https://github.com/vadimdemedes/ink) and [xterm.js](https://xtermjs.org/).

## What is this?

ink-web-ui provides ready-to-use terminal components for web applications. It bridges the gap between terminal interfaces and the browser — write terminal UIs with Ink's `Box` and `Text` primitives and render them in a real xterm terminal embedded in your web app.

## Tech Stack

- **Next.js** — App framework
- **ink-web** — Ink renderer for the browser
- **xterm.js** — Terminal emulator
- **Fumadocs** — Documentation
- **Tailwind CSS** — Styling

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to see it running.
